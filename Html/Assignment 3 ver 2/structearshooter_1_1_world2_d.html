Detailed Description

Essentially a set of application-wide global variables defining the dimensions of the "World2D" and the conversion factors from pixel to World2D units and back, as well as a few rendering constants. A cleaner version would replace all these public static variables by private instance variables accessed by setters & getters, so that we cann have multipe windows, each running a different World2D
